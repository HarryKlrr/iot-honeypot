# Results

Analysis outputs generated by [`../analysis/enhanced_credential_analyzer.py`](../analysis/enhanced_credential_analyzer.py) from the captured Cowrie logs.

## SSH brute-force

- `credential_analysis_run_1.png` — six-panel dashboard for Run 1 (184 attempts): top usernames, top passwords, success/failure split, attack timeline, source IPs, session distribution.
- `credential_analysis_run_2.png` — the same analysis for Run 2 (368 attempts).
- `run_comparison_ssh.png` — cross-run comparison: total attempts, success rate, username diversity, attack duration.
- `analyzer_console_output.png` — summary statistics printed by the analyser.

## Web probing

- `web_analysis_run_1.png` / `web_analysis_run_2.png` — six-panel web-attack dashboards: requested paths, HTTP status codes, tool attribution, request timeline, methods, error rate.
- `web_probing_findings.png` — headline web-probing metrics.
